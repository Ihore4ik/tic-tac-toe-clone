# TIC-TAC-TOE

##  A web application created with React + Vite!

## :rocket: Knowledges
 - `ReactJS`
 - `Redux Toolkit`

## :book: How to use

```
# Clone this repository
$ git clone https://github.com/Ihore4ik/tic-tac-toe-react-tailwind.git

# Go into the repository
$ cd tic-tac-toe-react-tailwind

# Install dependencies
$ npm install

# Run the app
$ npm run dev
```
